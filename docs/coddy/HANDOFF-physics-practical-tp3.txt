ANSWER KEYS (owner's priority - all independently re-derived and confirmed correct):
  - Weight W = mg: 0.650 kg -> 6.50 N; 2.5 kg -> 25 N; 0.300 kg -> 3.0 N; 0.250 kg -> 2.5 N. Mass is a
    scalar (same everywhere); weight is a vector, smaller where g is smaller (e.g. the Moon).
  - Density rho = m/V: 384 g / 480 cm^3 = 0.80 g/cm^3 (floats, < 1.0); 421.4 g / 490 cm^3 = 0.86.
  - Hooke: e = L - L_0 = 9.0 - 4.0 = 5.0 cm = 0.050 m; k = F/e = 2.0/0.050 = 40 N/m; load-extension
    graph gradient = k = 2.0 N / 4.0 cm = 0.50 N/cm = 50 N/m. Law fails past the limit of proportionality.
  - Moments: uniform rule, 2.0 N at the 20 cm mark, pivot 40 cm, centre of gravity 50 cm ->
    2.0 x (40-20) = W x (50-40) -> W = 4.0 N. Sum of clockwise = sum of anticlockwise; a weight above
    the pivot has zero moment.
  - Mass by moments: M = m(x/y) = 0.050 x 6.05/6.10 = 0.050 kg. Graphical: r/p = 0.5(q/p) + M/100, so
    gradient 0.5 (the 50g/100g ratio) and intercept M/100; intercept 0.90 -> M = 90 g (0.98 -> 98 g).
  - Friction: at constant velocity the resultant force is zero so pull = friction; W = Mg (0.300 ->
    3.0 N; 0.250 -> 2.5 N); mu = W/(L + mg) = 3.0/6.214 = 0.48; graph of pull vs load has gradient mu
    = 0.48 and intercept mu m g. mu is a ratio of forces, so dimensionless.
