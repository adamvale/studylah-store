ANSWER KEYS (owner's priority - all independently re-derived and confirmed correct):
  - Ohm: R = V/I = 12/0.50 = 24 ohms; 9.30/1.52 = 6.12 ohms. Component gradient 1/R = 0.050 A/V ->
    R = 20 ohms. R = rho L / A: longer -> more R, thicker -> less; doubling the area halves R.
  - Series/parallel: series 25 + 50 + 75 = 150 ohms (maximum); parallel 1/R = 1/25 + 1/50 + 1/75 =
    0.0733 -> R = 13.6 ohms (minimum). Series shares V by R; parallel shares I inversely.
  - Potential divider: V_out = (R2/(R1 + R2)) V = (6.0/10.0) x 5.0 = 3.0 V (and (6.0/9.0) x 9.0 = 6.0 V).
  - Power/energy: P = IV = 0.50 x 12 = 6.0 W; P = I^2 R = 2.0^2 x 5.0 = 20 W; E = VIt = 12 x 0.50 x
    120 = 720 J (2.0 min = 120 s); LED P = 0.117 x 0.36 = 0.042 W.
  - I-V: ohmic = straight line through the origin, gradient 1/R (R = 6.00 -> 0.167 A/V); filament lamp
    S-curve (non-ohmic); diode turn-on then steep (non-ohmic). Diode I = V_x/R = 22.0/22.0 = 1.00 A.
  - Wire experiment: R/L vs 1/L, gradient = R_X = (6.5 - 1.7)/(5.0 - 1.0) = 1.20 ohms, intercept =
    rho/A = 0.50 ohm/m.
  - Magnetism (ground truth, all verified): current out of the page -> ANTICLOCKWISE field (into the
    page -> clockwise); a solenoid's NORTH pole is the anticlockwise-current end; strengthen with more
    current / turns / a soft-iron core; Fleming's LEFT hand = the MOTOR force (thumb force, first
    finger field N-to-S, second finger current); Faraday = e.m.f. proportional to the rate of flux
    change; Lenz = the induced current opposes the change; Fleming's RIGHT hand = the generator's
    induced current.
